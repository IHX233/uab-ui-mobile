## 快速上手

### 安装

```bash
$ npm install --save uab-ui-mobile
# or
$ yarn add uab-ui-mobile
# or
$ pnpm add uab-ui-mobile
```

### 引入

```js
import { Button } from 'uab-ui-mobile'
```
